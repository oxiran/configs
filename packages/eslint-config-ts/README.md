# eslint-config-ts

> Shared typescript eslint config of my projects.

## Install
```shell
npm install --save-dev eslint @oxiran/eslint-config-ts
```

## Usage

Join in the `extends` of the .eslintrc file.

**.eslintrc**
```json
{
  "extends": ["@oxiran/eslint-config-ts"]
}
```

or **.eslintrc.js**
```javascript
module.exports = {
  extends: ['@oxiran/eslint-config-ts']
}
```
